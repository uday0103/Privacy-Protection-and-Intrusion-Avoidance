## 📌 Project Overview

This project presents a secure and efficient cloud-centric healthcare system for medical data sharing using cloudlet technology. The system ensures privacy protection, secure authentication, intrusion avoidance, and public verifiability for sensitive healthcare information.

The application enables secure communication between:

- IoT Devices
- Healthcare Centers
- Medical Cloud Servers
- Key Protection Servers (KPS)

---

## 🚀 Features

- Secure Medical Data Sharing
- User Authentication & Authorization
- Intrusion Avoidance Mechanism
- Public Verifiability
- Cloud-Based Healthcare Management

## 📖 How to Run Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/uday0103/Privacy-Protection-and-Intrusion-Avoidance.git
```

### 2️⃣ Configure Database
- Create MySQL database
- Import SQL file
- Update `connect.jsp` with database credentials

### 3️⃣ Run the Project
- Deploy using Apache Tomcat Server
- Open browser:

```bash
http://localhost:8080
```
